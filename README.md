## Getting Started (Local Setup)

1. Clone the repo:

   ```bash
   git clone https://github.com/Anikhe00/Quizzical.git
   ```

2. Open the folder:

   ```bash
   cd Quizzical
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

   - The app will be available at `http://localhost:3000`
